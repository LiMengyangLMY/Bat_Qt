## 功能
该应用程序为自动生成bat脚本的应用程序，可将一次性打开用户上传的文件路径对应的程序网页等。
### 1.添加所需文件
**添加文件路径窗口：**用户在该窗口添加需要打开的文件路径，包括：网页、可执行文件等。
**验证路径正确性按键：**保证用户上传的文件路径正确，即能否正确通过该路径打开文件。

### 2.显示文件路径列表

### 3.自动生成bat脚本
**生成bat脚本：**该脚本的功能为可以一次性直接打开用户上传的文件路径。
脚本示例：
```
chcp 65001 
::可正常显示中文
@echo off

start "" "path_name1"
start "" "path_name2"
start "" "path_name3"
```
